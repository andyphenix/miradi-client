Files - Viewing the Results

     

  On this tab you can check the results of your definition of the distribution tree.
  The tree shows all files that will be distributed in the generated media files.

You cannot remove files from this tree or add them to it. If you would like to remove a file that has been added with a directory entry, you have to use the excluded files and directories step or the excluded suffixes step in the files directory wizard. To exclude files and directories on a per-media set basis, please see the customize project defaults step in the media file wizard.

  On activating this tab, the file tree is re-read if the definition of the distribution tree has changed since the last time the file tree was shown. This background process can take a short while and is indicated by a "Please wait ..." entry in the result tree.

Should the contents of your hard disk have been modified in the meantime, you can use the [Refresh] button to re-read the displayed file tree.
