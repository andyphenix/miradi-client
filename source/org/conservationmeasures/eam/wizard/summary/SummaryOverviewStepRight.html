Navigation Hints for the Summary View and the Overall Application

Summary View

The first few "Summary View" tabs contain basic information fields that most projects will want to fill in about their project, team, scope, location, and planning. This basic information will be especially important when you export your project to various reports and databases. Some organizations also have their own custom data fields for their projects. You can find these by clicking on the appropriate tab.

Overall Application

All views have an interview pane on top and an action pane below. Use the previous and next buttons to move backwards and forward through the interview. You can resize or hide the interview pane by using the splitter and/or arrows. In many dialog boxes, you can use the directions button at the bottom left to go the appropriate interview page. Use the buttons on the upper right to get context sensitive additional information, examples, and workshop hints.

Note that the program automatically saves all changes so there is no "save" command. You can save a copy under a different name using the "Copy Project To" command.
