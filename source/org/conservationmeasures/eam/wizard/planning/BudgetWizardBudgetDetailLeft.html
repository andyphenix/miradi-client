At this point, you should have already developed your basic work plan showing activities/methods and tasks and assigning resources to each.

To add budget detail to an activity/method or task, click on the <Work Plan> radio button in the Planning View and select the item that you want to focus on. You should already see the number of units for each quarter for that resource. Using the drop down menu, you can then assign an accounting code and funding source for that resource.

Hint: The same resource may have different accounting codes or funding sources for different activities/ methods or tasks.

If you have already added the cost per unit for a resource, at this point you should see the total cost for that resource for the quarter and rolled up for the year. The annual amount is then displayed in the table at the top of the page. Click on the <Resources> tab and select the resource in the table if you need to edit the cost per unit information.

Hint: The current version of Miradi only supports financial expenses that can be assigned to a specific resource.

Hint: If you need to adjust the planning horizon for your project, go to the project start and end date fields in the Summary View.

Hint: You can edit the number decimal places for currency amounts in the Summary View.
