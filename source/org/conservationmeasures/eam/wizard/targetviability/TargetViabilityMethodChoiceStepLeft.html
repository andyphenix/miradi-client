Miradi currently supports two different viability analysis methods for assessing target status. Generally, within a project you should consistently use one method or the other, but you can use different methods on different targets if you wish.

To begin viability analysis on a target, bring up its properties dialog by double-clicking on the target ellipse in the diagram. Move and/or resize the dialog so these instructions remain visible. Then click one one of the buttons below to indicate which method you would like to use for this target.
