To use this tab, you have to first create a results chain in one of two ways:

  1. You can create a results chain from an existing strategy in your conceptual model by selecting that strategy and then using the <Create Results Chain> command (available on the toobar, the right mouse button menu, or in the actions menu) to create a results chain for it.
  2. You can create a blank results chain by using the <Create Results Chain> command without first selecting a strategy.
