Indicators can be assigned different priority ratings as follows:
  • Very High: Must be monitored.
  • High: High priority for monitoring.
  • Medium: Monitor only if resources allow.
  • Low: Not necessary to monitor.

You can use the <Edit Progress Report> button to update the status of a given indicator:

  • Scheduled for Future Implementation: Indicator is not yet started
  • Major Issues: Ongoing, but has major issues that need attention
  • Minor Issues: Ongoing, but has minor issues that need attention
  • On Track: Ongoing, generally on track
  • Completed: Successfully accomplished
  • Abandoned: No longer relevant or useful
