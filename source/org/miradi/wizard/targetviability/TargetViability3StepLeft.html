You should have already set the viability analysis mode for the target you are working on to "Key Attribute".

Hint:If you are trying to follow the interview and go through the steps in the Key Attribute Mode, it will be easier to work in the Viability View rather than in individual target dialogue boxes. However, once you stop using the interview, you can work either in the target dialogue box or in the Viability View.

The first step is to develop one or more Key Ecological Attributes (KEAs) for each target. To create a new KEA, click the <Create KEA> button. Select the new KEA in the upper panel, and then fill in the following fields in the lower panel:

  • ID: A user assigned identification code for the KEA.
  • Name: The name of the KEA. Generally 2-5 words (e.g., Shark Population Size).
  • Details: Detail about the KEA.
  • Type: The type of KEA.
  • Comments: Discussion about the KEA.
To edit or delete a KEA, click on the desired row in the upper panel, and then either edit the text fields in the lower panel, or click on the <Delete KEA> button.
