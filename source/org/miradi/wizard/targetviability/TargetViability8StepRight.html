The current status rating describes the indicator rating category where your key ecological attribute is today; the desired future status rating describes where you want to be in the future. The desired future status is a goal (or at least a sub-goal) for your project. You should consider the appropriate spatial extent and time frame for achieving the desired status; some changes may require long time periods (50-100 years).

If you are using the Simple Viability assessment method, then it will be necessary later on in the process to craft a goal for each target. If you are using the the Key Attribute viability assessment method, however, then technically all of your Future Status measurements for that target together represent the goal. Many projects in this situation still choose to develop an overarching goal for each target that summarizes the various desired future status measurements for all of that target's KEAs.
