Welcome to the Threat Rating View

Miradi's default method is the Simple Threat Rating system (see description of the two systems in the right wizard panel). If you want to use the Stress Based Rating system, from the <Edit/Preferences> menu, select the <Threat Rating View> tab. In this tab, change the Threat Rating Mode field to "Stress Based Threat Rating Mode."

The table below contains a summary of your threat ratings. Target-Threat combinations that have been rated should appear colored in. If there is a threat or target that is missing, click here to return to the diagram view to add it. If a threat is not linked to a specific target, you can create this by right (alt) clicking on a blank cell in the table (note that this also creates a corresponding link in the diagram).

Click next to continue with the threat rating process.
