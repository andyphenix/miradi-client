To insert a direct threat, select a target and then click the <Insert Direct Threat> button located on the Control Bar. The software will add a new direct threat linked to the target and open a dialog box.

Type type in the Name of your direct threat, and if desired, an ID (see hint below). You can also add additional information about the threat in the Details and/or Comments fields. You should also pick the entry that best describes your threat from the Standard Classification field that lists a taxonomy of all direct threats in the drop-down menu.

Using the mouse or arrow keys, move the threat to wherever you want it to be.

Hint: The Direct Threat ID field can be used for a short code used to identify and sort your direct threats. One common system is to assign the letters DT and a number to each Direct Threat (e.g., DT1, DT2, etc.)

Hint: In most Miradi dialogue boxes, the Details field is used to record a longer description of the factor, whereas the Comments field is meant for discussion about the factor.
