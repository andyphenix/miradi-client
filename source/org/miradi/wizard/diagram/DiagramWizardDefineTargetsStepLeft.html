Next, you need to add all of your project's conservation targets to your scope. You should already have one target in your Scope box. You can double click on the green oval representing this target and type in the Name of your target, and if desired, an ID (see hint below). You can also add additional information about the target in the Details and/or Comments fields. If it is a species target, you can also record the Latin Name.You can also use the Habitat Association field to record the standard name of the system in which a species target is found, or for an ecosystem target itself.

To add additional targets, click on the <Insert Target> button, select the target and move it to wherever you want it to be, and the double click on it to enter information into the dialogue box.

Hint: The Target ID field can be used for a short code used to identify and sort your targets. One common system is to assign a letter to each Target (e.g., A. Coral Reefs, B. Sharks, etc.).

Hint: In most Miradi dialogue boxes, the Details field is used to record a longer description of the factor, whereas the Comments field is meant for discussion about the factor.
