Just as each strategy will have one or more activities associated with it, each monitoring indicator will also have one or more methods associated with it.

In the Planning View, click on the <Work Plan> radio button and select an indicator that you want to focus on. Think about the specific methods that you will have to undertake to measure this indicator. Click on the <Create Method> button to add a new method and then type the name of the method in the label box in the bottom panel. If necessary, you can have multiple methods for each indicator. Repeat for other indicators as needed. You can use the <Move Item Up> and the <Move Item Down> buttons to change the order.

If necessary, you can also highlight a method and click on the <Create Task> button to add tasks under this method. (If you are really detail oriented, you can even create sub-tasks and sub-sub tasks under a task.)
