Summary Threat Ratings - Miradi rolls up threat ratings for each target and threat using another rule-based system for combining ratings for each target-threat combination. Click here for a detailed description of this system.

New Feature Hint: Direct threats do not need to be directly linked to targets to have target-threat ratings. If, however, a target-threat combination should not be rated, click the N/A - not a direct link checkbox located next to the Target-Threat Rating. Click the ? icon next to the checkbox for more information.
