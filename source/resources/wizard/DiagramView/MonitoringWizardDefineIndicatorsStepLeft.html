Indicators are linked to factors in a model. To create a new indicator, double click on a factor and then click on the <Indicators> tab in the properties dialogue box. Click on the <Create Indicator> button and fill in the following fields:
  • Indicator ID: A user assigned identification code for the indicator (e.g., 3, 17).
  • Indicator Name: The name of the indicator. Generally 2-5 words (e.g., Number of Sharks).
  • Details: Details about the indicator.
  • Comments: Discussion about the indicator.

Each indicator will have one or more methods associated with it. To assign a method, use the <Edit Methods> button the bottom of the indicators panel. Fill in the label field at the bottom of the page. You can either create a new method or share an existing method. If necessary, you can add multiple methods for each indicator.

The next step is to determine the measurement for each indicator. Select an indicator in the table and then click the <Create Measurement> button. Next in the bottom panel, fill in the following fields:

  • Measurement Date: The date of the most recent indicator measurement.
  • Measurement Value: The current value or summary qualitative description of the indicator.
  • Details: Full description of the current value or qualitative description of the indicator.
  • Trend: Your assessment of the direction that this indicator is heading at the moment of the current measurement.
  • Source: Your confidence rating in the source of the indicator measurement.
  • Comments: Discussion about the measurement.

Next, determine the desired future status for the indicator. In the upper table, click on the blue oval beneath the indicator you are working on and then edit the fields in the bottom panel.

For the selected indicator's desired future value, fill in the following fields:

  • Desired Future Value Date: The date at which the indicator value will be assessed.
  • Desired Future Value: The desired future value or summary qualitative description of the indicator.
  • Details: Full description of the current value or qualitative description of the indicator.
  • Comments: Discussion about the desired future status.

    Miradi will automatically display the Future Status value, along with a blue oval icon, in the cell of the monitoring table corresponding to the desired future status.

    If you want to use an existing indicator from another factor, click on the <Clone Indicator> button and follow the instructions. To edit or delete an indicator, click on the desired row and then either edit the text in the box at the bottom or click on the <Delete Indicator> button.

    You can also view and edit all your project's indicators using the "Indicators Only" setting in the <Planning View>.

    Hint: If you used the detailed method of viability analysis, you have already created indicators for your targets. The information on this page is not applicable for targets set to use the detailed method. However, you should create indicators for other factors, where appropriate.
