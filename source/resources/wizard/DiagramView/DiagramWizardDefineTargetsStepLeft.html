Next, you need to add your project's conservation targets to your scope.

Click on the <Insert Target> button and you will get a dialogue box into which you can type in the Name of your target, and if desired, an ID (see hint below). You can also add additional information about the target in the Details and/or Comments fields. If it is a species target, you can also record the Latin Name. You can also use the Habitat Association field to record the standard name of the system in which a species target is found, or for an ecosystem target itself.

Once you close the dialogue box, you can select the target and move it wherever you would like it to be on your diagram (typically within the scope box). You can also edit the target's name and other properties by double clicking on the target oval.

Repeat as needed to enter all your conservation targets.

Hint: The Target ID field can be used for a short user-defined code used to identify and sort your targets. One common system is to assign a letter to each Target (e.g., A. Coral Reefs, B. Sharks, etc.).

Hint: In most Miradi dialogue boxes, the Details field is used to record a longer description of the factor, whereas the Comments field is meant for discussion about the factor.
