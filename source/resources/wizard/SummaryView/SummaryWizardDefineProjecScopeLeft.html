Click on the <Scope> tab and add your project's scope to the appropriate fields. The Scope Name field should contain a 3-5 word summary title of your scope. The Scope Text field should contain the complete description of your project's scope.

Click next to define your project's vision.

You can also fill out the remaining fields on this page if desired, especially if this information will be helpful to have in reports generated from your Miradi file.

The first set of fields describes the Biodiversity Features of your project site:

  • Biodiversity Area is used to record the area of the biological systems of interest at your project site. You can also record any Notes about the area.
  • IUCN Red List Species is used to list any endangered or threatened species found at your project site.
  • Other Notable Species can be used to record any additional species that you want to note.

The second set of fields describes the Human Stakeholders at your project site:

  • Human Stakeholder Pop Size refers to the number of people within the scope of the project, while the Notes field can be used to record any detail about how you define these stakeholders.
  • Social Context can be used to record any additional information about the human systems at the site.

The final set of fields can be used to describe the features of your site if it is a formal Protected Area.
