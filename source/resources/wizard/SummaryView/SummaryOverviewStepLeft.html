Welcome to the Miradi Interview & the Summary View!

As you go through the steps in the CMP Open Standards, this leftside panel of the interview will provide you with step-by-step instructions. The rightside panel will provide definitions and guidance.

The <Project> tab below provides an overview of your project. You can fill in your Project Name, the Project Data Effective Date (the date of your most recent substantive updating of the Miradi project file), and your Project's Website address in the appropriate fields.

Hint:Note that project file name is read only as shown by the grey background. To edit this field, you have to use the file manager.

Click next to define your initial project team.

Later on, you can return to this page and fill in the remaining fields. The Project Description field can be used for a summary or abstract of your project. The Project Status field can be used to generally describe how your project is doing at the moment. Finally, the Next Steps field can be used to describe any critical next steps; this field is especially useful when you want to send your project file to other team members.
