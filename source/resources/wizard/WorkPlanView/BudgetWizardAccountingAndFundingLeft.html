To develop a budget, you will first want to create accounting codes and funding sources.

In the <Accounting Codes> tab at the top of the Work Plan View, if your project already has accounting codes from your home organization, you can use the <Import Accounting Codes> button to import a tab delimited text file (2 columns, code followed by description). If you have to create codes from scratch, click on the <Create Accounting Code> button to add a new one and then type the code and the label in the box at the bottom of the page. Repeat for other codes as needed.

Miradi also supports two additional Categories that can be used to further classify expenses. This can be used to denote a matching contribution, or an extra level of detail in your accounting systems for example. Category #1 and Category #2 are managed in the same way as Funding Sources.

On the <Funding Sources> tab at the top of the Work Plan View, click on the <Create Funding Source> button to add a new one and then type the code (if needed) and the label in the box at the bottom of the page. Repeat for other funding sources as needed.
