Each strategy will have one or more activities associated with it.

In the Work Plan View, select the <Work Plan> tab and select a strategy that you want to focus on in the tree. Think about the specific activities that you will have to undertake to carry out this strategy. Click on the <Create...> button and select <Create Activity> and then type the name of the activity in the item field in the bottom panel. Repeat this for other activities needed to implement this strategy as needed.

If necessary, you can also highlight an activity and click on the <Create...> button and select <Create Task> to add tasks under this activity.
