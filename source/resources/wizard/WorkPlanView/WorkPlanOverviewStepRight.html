Hints for the Work Plan View

You can specify which Conceptual Model or Results Chain diagram to display in the <Work Plan> tab by selecting from the Diagrams - include data from and the Display this selection drop down choosers. The work plan table will display the strategies, activities and tasks from the strategies that are in the selected diagram(s).

You can also specify whether to display only Action rows (strategies and yellow activities) or Monitoring rows (strategies and purple monitoring activities) by clicking the <Customize Table...> button and making a selection in the Include Data From drop down menu. The project totals values will only reflect the data that is being displayed.

Click on the <Work Plan> tab to see an overview of your plan. In the tree, clicking on the + and - symbols enables you to expand or collapse different parts of the hierarchy. Click the <Expand To...> button to select the branches of the tree to display. To display the Assignment and Expense rows, click the <Customize Table...> button, and check the appropriate Rows check boxes.

Expenses are estimated in the default project currency. To set the currency, go to the <Settings> tab in the Work Plan View (or go to Edit->Preferences, and select the Planning tab). Select the <Financial Settings> side tab to adjust the currency type and the number of decimal places displayed.

Hint: The values in the Who Work Assignments column roll up from lower levels in the hierarchy so that it is easy to see all the people that will be required for a given strategy or activity.

Hint: The values in the Timeline column are entered from the <Timeline> side tab in the sub-panel of an action, and are independent of the Work Unit or Expense entries in the Work Plan table.

Hint: The values in the When Work Assignments column are based on the Work Unit entries in the Work Plan table.
