Future versions of Miradi will allow you to generate reports on actual expenditures across different time periods by accounting code, activity, funding source, and against budgets.

Currently, it is possible to review all work and expenses that have been assigned to a given Project Resource, Accounting Code or Funding Source for a given time period. This should aid in your planning by helping to determine your Resource and Funding needs per fiscal quarter or year.

Hint:You can create individual work plans by using the <Filter Project Resources...> button and selecting the resources you want to view. This will update the Work Plan to only show work days and budget totals for the selected resource(s).
