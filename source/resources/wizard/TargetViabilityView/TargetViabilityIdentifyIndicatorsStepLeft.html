The next step is to develop one or more indicators for each KEA.

To create a new indicator, make sure the KEA is selected in the upper panel, and click the <Create Indicator> button.  Fill in the following fields:

  • Indicator ID: A user assigned identification code for the indicator.
  • Indicator Name: The name of the indicator. Generally 2-5 words (e.g., Number of Sharks).
  • Details: Details about the indicator.
  • Comments: Discussion about the indicator.
