For Simple viability analysis, double click on the target that you want to work on and make sure the Viability Analysis Mode field is set to "Simple".

Next, select the Status side tab and then use the drop down box in the Target Viability Status field to assign a status rating for the target. This rating will appear at the top of the properties dialog, and will be displayed in the diagram as a small oval next to the target. You should also select the Details side tab and fill out the Current Status Justification field with any information about how you arrived at this rating.

Goals and indicators for this target will be identified and added later.
