What's New in Miradi 4.0

Miradi 4.0 contains many improvements and enhacements. In addition to lots of "behind the scenes" technical improvements, some major highlights include:

  • New Integrated .Miradi File Format - Miradi 3.x stored data in sub-folders in the Miradi directory on your computer. If you wanted to transfer the file, you had to export an .mpz file. Now, project data are stored in your Miradi directory in one file that uses the .Miradi extension. You can treat these .Miradi files as you would any other file (e.g. .doc or .pdf) on your computer. Click here for more details.
  • Ability to Edit Directly in Upper Tables - You can now edit any blue text that you see in tables without having to click on the lower panel.
  • Support for Text Formatting - You can now use the right (alt) mouse button to add text formatting (e.g. bold, italic) and hyperlinks to any multi-line text field.
  • Designate Task Leader - You can now assign responsibility for leading an activity or task in work planning mode. You can also make work assignments from diagram view.
  • Support for XSLT Report Generation - Miradi now supports the generation of custom reports using the XSLT language.
  • Support for Miradi Dashboard and Miradi Share Online Systems - Miradi Desktop software is now designed to readily export information to the Miradi Dashboard application and the Miradi Share online system for managing conservation programs. See the miradi.org website for more details.
