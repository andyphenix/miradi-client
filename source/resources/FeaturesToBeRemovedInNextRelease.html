Features to be Removed or Revised in Next Release

The following features are scheduled to be removed or revised in the next release. Please contact feedback@miradi.org if you have questions or concerns about the removal of a feature.

Summary view / Planning: The content of this tab will continue to be available (1) in the Settings tab of the Work Plan view and in the Planning tab of Miradi Preferences (select Edit / Preferences from the main menu).

Summary view / Team, Other Orgs, Scope, Location: The content of these tabs will be managed in Miradi Share.
