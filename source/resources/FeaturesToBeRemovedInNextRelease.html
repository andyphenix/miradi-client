Features to be Removed in Next Release

The following features are scheduled to be removed in the next release. Please contact feedback@miradi.org if you have questions or concerns about the removal of a feature.

Summary view / Planning: The content of this tab will continue to be available (1) in the Settings tab of the Work Plan view and in the Planning tab of Miradi Preferences (select Edit / Preferences from the main menu).

Summary view / TNC, WWF, WCS, RARE, FOS: These tabs store legacy data that is presumed to be no longer necessary.

Reports view / Standard Reports: The content in the standard reports will continue to be available as selections for report templates created at the Report Templates tab.

Dashboard: This feature is presumed to be unused.
