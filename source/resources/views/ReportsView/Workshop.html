Exporting Workshop Data in Miradi to Other Applications

Although you may not want to use Miradi to display information in a workshop, it is a very useful way to capture the information put on a sticky board or a flip chart so that it can be documented and shared with other workshop participants.

Exporting Data in Miradi to Other Applications

There are many ways to export your information from Miradi to other applications.

A. EXPORT CURRENT PAGE AS...

This command is useful to export a specific image or table for use in a document or presentation. Options include:

Export JPG or PNG Images

Use the <File/Export Current Page as.../JPEG Image> or <File/Export Current Page as.../PNG Image> menu commands to capture a specific diagram or table on your screen in an image file. This method is particularly useful to:
  • Put a diagram or table in a document or presentation
  • Create an image that can be put in a website

Export Rich Text Format (RTF) File

Use the <File/Export Current Page as.../RTF File> menu command to export the currently viewed table to a Rich Text Format (RTF) file. This method is particularly useful to:
  • Export a table or other information in a format that can be edited by a word processor
  • Export pieces of a custom report

Export Tab-Delimited Data

Use the <File/Export Current Page as.../Tab-Delimited Data> menu command to export the currently viewed table to a tab-delimited file. This method is particularly useful to:
  • Export the current configuration from your planning view to a spreadsheet like Excel
  • Export financial information

B. EXPORT CURRENT PROJECT AS...

This command is useful to export your entire project file so that it can be shared with other users. Options include:

Export Miradi Zip (MPZ) File

Use the <File/Export Current Project as.../Miradi Project Zip (MPZ) File> menu command to capture your entire project in a small zip file. This method is particularly useful to:
  • Send your project to another Miradi user
  • Create a backup copy of your project

Export ConPro (CPMZ) File (CP

Use the <File/Export Current Project as.../ConPro (CPMZ) File> menu command to capture your entire project in a zip file that can be imported into The Nature Conservancy's ConPro database. Note that if your organization would like to import Miradi data into your own database, contact support@miradi.org. This method is particularly useful to:
  • Upload your information to TNC's ConPro database
  • Share your project information with other practitioners

Export Project XML File

Use the <File/Export Current Project as.../Project XML File> menu command to create an XML file with all your project data. Although managing the XML file will require technical skills, this method is particularly useful to:
  • Create a file that can be imported by a database
  • Export all of your project's information into formats that can be read by other applications

C. GENERATE REPORTS

Use the <Reports> view to create a Rich Text Format (RTF) file that contains various configurations of your project data. This method is particularly useful to:
  • Create a document that can be edited in a Word Processor
  • Create a standard report for a donor or other key audience for your project
