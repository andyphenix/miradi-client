Miradi stores active files in a special Miradi folder on your computer. In different operating systems, these are:

  • Windows XP: C:\Documents and Settings\Your User Name\My Documents\Miradi
  • Windows Vista: C:\Users\Your User Name\Documents\Miradi
  • Windows 7: C:\Users\Your User Name\Documents\Miradi
  • Mac OSX: /Users/yourname/Miradi
  • Linux: /home/Your User Name/Miradi

In this folder, you will see one file for each Miradi project (with a .Miradi extension). You can rename, delete, copy, or move these files using your operating system's standard commands. You can also manage these files from the initial Welcome (No-Project) page of the Miradi program.

In some instances, you may need to change the default location for your Miradi files. For example, if your Windows laptop normally stores "My Documents/Documents" on a network drive, but you then disconnect your computer from the network, you will need to specify a new default folder on your local hard drive. Click on the button below to make this change.

Hint: Make sure that the folder that you specify is included in the backup routine for your computer to ensure that you do not lose all your Miradi work in the event of a system failure.
